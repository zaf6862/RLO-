Write a method called chopN that takes an array of Strings called inputArr and an int called chopNum.

The method returns an array where each member of the returned array has chopNum length.

Any String in inputArr that has length < chopNum is not chopped and is included in the returned array. Assume all Strings in inputArr have length >= 1.

For example, when this code is executed:

String[] names = {"Ali", "Zhichao", "Mohit", "Shahrzad", "Aaron", "Celeste", "Bo"};

String[] resultArr = chopN(names, 3);

The contents of resultArr would be these Strings: "Ali", "Zhi", "Moh", "Sha", "Aar", "Cel", "Bo"

At a more advanced level, arrays can be used to represent the mathematical object matrices and we can use arrays to perform mathematical operations.
